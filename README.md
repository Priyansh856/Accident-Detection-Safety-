# Accident-Detection-Safety-
The Accident Detection &amp; Safety App is designed to enhance personal safety by leveraging mobile sensor data such as the accelerometer and gyroscope to identify sudden impacts or abnormal motion. In case of a suspected accident, the app triggers an emergency protocol that sends an SOS message along with the user’s live location to predefined.

✨ Key Features
	•	🚀 Accident Detection System
	•	Detects sudden impacts using accelerometer & gyroscope
	•	Intelligent logic to reduce false positives
	•	⏱️ Smart Alert Confirmation
	•	10-second countdown before triggering emergency alerts
	•	User can cancel alert if safe
	•	📍 Real-Time Location Sharing
	•	Sends accurate GPS location via Google Maps link
	•	📩 Emergency SOS Alerts
	•	Automatically sends SMS to emergency contacts
	•	Optional auto-call feature
	•	👥 Emergency Contact Management
	•	Add, edit, and delete trusted contacts
	•	🔋 Background Monitoring
	•	Runs continuously using foreground services
	•	🎨 Modern UI
	•	Clean and user-friendly interface with Material Design

	1.	The app continuously monitors sensor data in the background.
	2.	When a sudden impact is detected, it assumes a possible accident.
	3.	A confirmation screen appears with a countdown timer.
	4.	If the user does not respond, the app:
	•	Fetches the current location
	•	Sends an SOS message to emergency contacts
